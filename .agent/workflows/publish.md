# Publish to GitHub Workflow

After making any code changes to the E-AI project, follow these steps to push to GitHub:

// turbo-all

1. Stage all changes:
```bash
git add -A
```

2. Commit with a descriptive message:
```bash
git commit -m "[describe the changes made]"
```

3. Push to GitHub:
```bash
git push origin main
```
